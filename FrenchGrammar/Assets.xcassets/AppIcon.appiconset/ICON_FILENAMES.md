To use the Leek French cover as the Home Screen icon, resize the cover image to these PNG filenames in this folder:

- Icon-20@2x.png: 40x40
- Icon-20@3x.png: 60x60
- Icon-29@2x.png: 58x58
- Icon-29@3x.png: 87x87
- Icon-40@2x.png: 80x80
- Icon-40@3x.png: 120x120
- Icon-60@2x.png: 120x120
- Icon-60@3x.png: 180x180
- Icon-1024.png: 1024x1024

After adding the files, add each filename to the matching entry in `Contents.json`.
